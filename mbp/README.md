# MacBook Pro


## Initial set up

1. Install git

```bash
sudo apt-get install git-core
```

2. Clone this repo at the home folder

```bash
cd ~  
git clone git://github.com/patriciogonzalezvivo/.dotfiles.git
cd .dotfiles
```

3. Run Install Core Apps

```bash
./install_core.sh 
./install_tools.sh
./install_net.sh
./install_python.sh
```
